API reference for Angular Material paginator

import {MatPaginatorModule} from '@angular/material';

Services

MatPaginatorIntl

To modify the labels and text displayed, create a new instance of MatPaginatorIntl and include it in a custom provider

Properties
NameDescription

changes

Type: Subject<void>

Stream that emits whenever the labels here are changed. Use this to notify components if the labels have changed after initialization.

getRangeLabel

Type: (page: number, pageSize: number, length: number) => { if (length == 0 || pageSize == 0) { return `0 of ${length}`; } length = Math.max(length, 0); const startIndex = page * pageSize; // If the start index exceeds the list length, do not try and fix the end index to the end. const endIndex = startIndex < length ? Math.min(startIndex + pageSize, length) : startIndex + pageSize; return `${startIndex + 1} - ${endIndex} of ${length}`; }

A label for the range of items within the current page and the length of the whole list.

itemsPerPageLabel

Type: 'Items per page:'

A label for the page size selector.

nextPageLabel

Type: 'Next page'

A label for the button that increments the current page.

previousPageLabel

Type: 'Previous page'

A label for the button that decrements the current page.

Directives

MatPaginator

Component to provide navigation between paged information. Displays the size of the current page, user-selectable options to change that size, what items are being shown, and navigational button to go to the previous or next page.

Selector: mat-paginator

Exported as: matPaginator
Properties
NameDescription
@Input()

length

Type: number

The length of the total number of items that are being paginated. Defaulted to 0.

@Input()

pageIndex

Type: number

The zero-based page index of the displayed list of items. Defaulted to 0.

@Input()

pageSize

Type: number

Number of items to display on a page. By default set to 50.

@Input()

pageSizeOptions

Type: number[]

The set of provided page size options to display to the user.

@Output()

page

Type: new EventEmitter<PageEvent>()

Event emitted when the paginator changes the page size or page index.

Methods
hasNextPage

Whether there is a next page.

hasPreviousPage

Whether there is a previous page.

nextPage

Advances to the next page if it exists.

previousPage

Move back to the previous page if it exists.

Additional classes

PageEvent

Change event object that is emitted when the user selects a different page size or navigates to another page.

Properties
NameDescription

length

Type: number

The current total number of items being paged

pageIndex

Type: number

The current page index.

pageSize

Type: number

The current page size
