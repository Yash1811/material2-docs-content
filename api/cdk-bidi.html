API reference for Angular CDK bidi

import {BidiModule} from '@angular/cdk';

Services

Directionality

The directionality (LTR / RTL) context for the application (or a subtree of it). Exposes the current direction and a stream of direction changes.

Properties
NameDescription

change

Type: new EventEmitter<void>()

Stream that emits whenever the 'ltr' / 'rtl' state changes.

value

Type: Direction

The current 'ltr' or 'rtl' value.

Directives

Dir

Directive to listen for changes of direction of part of the DOM. Provides itself as Directionality such that descendant directives only need to ever inject Directionality to get the closest direction.

Selector: [dir]

Exported as: dir
Properties
NameDescription
@Output('dirChange')

change

Type: new EventEmitter<void>()

Event emitted when the direction changes.

value

Type: Direction

Current layout direction of the element.
