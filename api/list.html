MdListOption

Component for list-options of selection-list. Each list-option can automatically generate a checkbox and can put current item into the selectionModel of selection-list if the current item is checked.

Selector: md-list-option
Properties
NameDescription
@Input()

disableRipple

Whether the ripple effect on click should be disabled. This applies only to list items that are part of a selection list. The value of disableRipple on the md-selection-list overrides this flag

@Input()

checkboxPosition

Whether the label should appear before or after the checkbox. Defaults to 'after'

@Input()

disabled

Whether the option is disabled.

@Input()

value

@Input()

selected

onFocus

Emitted when the option is focused.

@Output()

selectChange

Emitted when the option is selected.

@Output()

deselected

Emitted when the option is deselected.

@Output()

destroyed

Emitted when the option is destroyed.
