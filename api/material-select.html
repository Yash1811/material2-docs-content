API reference for Angular Material select

import {MatSelectModule} from '@angular/material';

Directives

MatSelectTrigger

Allows the user to customize the trigger that is displayed when the select has a value.

Selector: mat-select-trigger

MatSelect

Selector: mat-select

Exported as: matSelect
Properties
NameDescription
@Input('aria-label')

ariaLabel

Type: string

Aria label of the select. If not specified, the placeholder will be used as label.

@Input('aria-labelledby')

ariaLabelledby

Type: string

Input that can be used to specify the aria-labelledby attribute.

@Input()

compareWith

Type: (o1

A function to compare the option values with the selected values. The first argument is a value from an option. The second is a value from the selection. A boolean should be returned.

@Input()

disableRipple

Type: boolean

Whether ripples for all options in the select are disabled.

@Input()

errorStateMatcher

Type: ErrorStateMatcher

An object used to control when error messages are shown.

@Input()

id

Type: string

Unique id of the element.

@Input()

multiple

Type: boolean

Whether the user should be allowed to select multiple options.

@Input()

panelClass

Type: string|string[]|Set<string>|{[key: string]: any}

Classes to be passed to the select panel. Supports the same syntax as ngClass.

@Input()

placeholder

Type: string

Placeholder to be shown if no value has been selected.

@Input()

required

Type: any

Whether the component is required.

@Input()

value

Type: any

Value of the select control.

@Output()

change

Type: EventEmitter<MatSelectChange>

Event emitted when the selected value has been changed by the user.

@Output()

onClose

Type: EventEmitter<void>

Event emitted when the select has been closed.

@Output()

onOpen

Type: EventEmitter<void>

Event emitted when the select has been opened.

controlType

Type: 'mat-select'

A name for this control that can be used by mat-form-field.

customTrigger

Type: MatSelectTrigger

User-supplied override of the trigger element.

empty

Type: boolean

Whether the select has a value.

errorState

Type: boolean

Whether the select is in an error state.

focused

Type: false

Whether the select is focused.

ngControl

Type: NgControl

optionGroups

Type: QueryList<MatOptgroup>

All of the defined groups of options.

optionSelectionChanges

Type: Observable<MatOptionSelectionChange>

Combined stream of all of the child options' change events.

options

Type: QueryList<MatOption>

All of the defined select options.

overlayDir

Type: ConnectedOverlayDirective

Overlay pane containing the options.

panel

Type: ElementRef

Panel containing the select options.

panelOpen

Type: boolean

Whether or not the overlay panel is open.

selected

Type: MatOption | MatOption[]

The currently selected option.

shouldPlaceholderFloat

Type: boolean

stateChanges

Type: new Subject<void>()

Stream that emits whenever the state of the select changes such that the wrapping MatFormField needs to run change detection.

trigger

Type: ElementRef

Trigger that opens the select.

triggerValue

Type: string

The value displayed in the trigger.
