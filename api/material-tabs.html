API reference for Angular Material tabs

import {MatTabsModule} from '@angular/material';

Directives

MatTab

Selector: mat-tab

Exported as: matTab
Properties
NameDescription
@Input('label')

textLabel

Type: string

The plain text label for the tab, used when there is no template label.

content

Type: TemplatePortal<any> | null

isActive

Type: false

Whether the tab is currently active.

origin

Type: number | null

The initial relatively index origin of the tab if it was created and selected after there was already a selected tab. Provides context of what position the tab should originate from.

position

Type: number | null

The relatively indexed position where 0 represents the center, negative is left, and positive represents the right.

templateLabel

Type: MatTabLabel

Content for the tab label given by.

MatTabLabel

Used to flag tab labels for use with the portal directive

Selector: [mat-tab-label] [matTabLabel]

MatTabNav

Navigation component matching the styles of the tab group header. Provides anchored navigation with animated ink bar.

Selector: [mat-tab-nav-bar]

Exported as: matTabNavBar, matTabNav
Properties
NameDescription
@Input()

Type: ThemePalette

Background color of the tab nav.

disableRipple

Type: boolean

Whether ripples should be disabled for all links or not.

MatTabGroup

Material design tab-group component. Supports basic tab pairs (label + content) and includes animated ink-bar, keyboard navigation, and screen reader. See: https://www.google.com/design/spec/components/tabs.html

Selector: mat-tab-group

Exported as: matTabGroup
Properties
NameDescription
@Input()

backgroundColor

Type: ThemePalette

Background color of the tab group.

@Input()

dynamicHeight

Type: boolean

Whether the tab group should grow to the size of the active tab.

@Input()

headerPosition

Type: MatTabHeaderPosition

Position of the tab header.

@Output()

Output to enable support for two-way binding on [(selectedIndex)]
